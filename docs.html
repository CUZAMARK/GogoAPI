Gogo Usage Documentation

Usage

Use the following endpoints to access the Gogo API.
  • /api/gogo/docs - This page
  • /api/gogo/home - Get Recent
  • /api/gogo/search/ - Search gogo for the term
  • /api/gogo/details/ - Returns details for the slug in the url
  • /api/gogo/seasons/ - Gives {"seasons":[["not available","season_id"]]} as response
  • /api/gogo/episodes/ - Fetches the episodes for the specified season_id
  • /api/gogo/source/ - Fetches the m3u8 for the requested episode
  • /api/gogo/trending/ - Returns the popular animes across specified timeline/ across wekk when not specfied
  • /api/gogo/popular/ - This route returns the popular animes in similar format as home but with more data
  • /api/gogo/tv/ - Returns newly added tv series to the site
  • /api/gogo/movie/ - Returns newly added movies to the site

Home

Get the recent episodes
Endpoint: /api/gogo/home
Method: GET
Parameters:
  • None
Example:
GET: /api/gogo/home
Response: {"Latest Subbed":[{"image":"image url","url":"slug","title":"name of anime","type":"SUB","rating":"?"},{...}...],"Latest Dubbed":[...],"Latest Chinese":[...]}

Search

Search gogo for the term
Endpoint: /api/gogo/search/{term}
Method: GET
Parameters:
  • term -> The term to search for
Example:
GET: /api/gogo/search/one%20piece
Response: {"Search Results Subbed":[{"image":"image url","url":"slug","title":"name of anime","type":"SUB","rating":"?"},{...}...],"Search Results Dubbed":[{...},...]}

Details

Get details for the slug in the url
Endpoint: /api/gogo/details/{slug}
Method: GET
Parameters:
  • slug -> The slug to get details for
Example:
GET: /api/gogo/details/category/one-piece
Response: {"title":"One Piece","image":"image","released":"TV Series","description":"details","genre":[["https://gogoanimehd.io/genre/action","Action"],[...]...]year":"1999","time_seasons":"Ongoing"}

Seasons

Get the seasons for the slug in the url
Endpoint: /api/gogo/seasons/{slug}
Method: GET
Parameters:
  • slug -> The slug to get seasons for
Example:
GET: /api/gogo/seasons/one-piece
Response: {"seasons":[["not available","171"]]}

Episodes

Get the episodes for the slug in the url
Endpoint: /api/gogo/episodes/{slug}/{season_id}
Method: GET
Parameters:
  • slug -> The slug to get episodes for
  • season_id -> The season_id to get episodes for
Example:
season_id = 171 (from above example)
GET: /api/gogo/episodes/one-piece/171
Response: [["one-piece-episode-1083","Episode 1083"],["one-piece-episode-1082","Episode 1082"],[...]...]
Explanation of response: [["slug","title"],["slug","title"],[...]...]

Sources

Get the sources for the slug in the url
Endpoint: /api/gogo/source/{slug}/{episode_slug}
Method: GET
Parameters:
  • slug -> The slug to get sources for
  • episode_slug -> The episode_slug to get sources for
Example:
episode_slug = one-piece-episode-1083 (from above example)
GET: /api/gogo/source/one-piece/one-piece-episode-1083
Response: {"source":"main_m3u8_file","alt_src":"alternative_m3u8_file","thumbnails":[],"subs":[],"unknown":[],"title":"{anime name} {episode title}"}

Trending

Get the trending animes
Endpoint: /api/gogo/trending?timeline=[week,month,all][default:week]
Method: GET
Parameters:
  • timeline -> The timeline for which we need to get trending possible values = [month, week, all]
Example:
GET: /api/gogo/trending?timeline=month
Response: [{"title": "...", "url": "...", "released_ep_count": "..."}...]

Popular

Get the popular animes
Endpoint: /api/gogo/popular?page=[int][default=1]
Method: GET
Parameters:
  • page -> get older data by increasing page number
Example:
GET: /api/gogo/popular
Response: [{"title": "...", "url": "...", "image": "...", "genres": [["url", "genre_name"]...]}...]

Movie

Get the latest anime movies added
Endpoint: /api/gogo/movie
Method: GET
Example:
GET: /api/gogo/movie
Response: {"Newly Added Movies":[{"image":"image url","url":"slug","title":"name of anime","type":"SUB","rating":"?"},...]}

Tv

Get the latest anime movies added
Endpoint: /api/gogo/tv
Method: GET
Example:
GET: /api/gogo/tv
Response: {"Newly Added Tv":[{"image":"image url","url":"slug","title":"name of anime","type":"SUB","rating":"?"},...]}
